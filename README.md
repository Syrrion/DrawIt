# DrawIt - Jeu de dessin multijoueur

## Installation

1. Assurez-vous d'avoir [Node.js](https://nodejs.org/) installé sur votre machine.
2. Ouvrez un terminal dans le dossier du projet.
3. Installez les dépendances :
   ```bash
   npm install
   ```

## Lancement

1. Démarrez le serveur :
   ```bash
   npm start
   ```
2. Ouvrez votre navigateur à l'adresse : [http://localhost:3000](http://localhost:3000)

## Comment jouer

1. Choisissez un pseudo et personnalisez votre avatar.
2. Créez une room privée ou rejoignez-en une existante avec un code.
3. Partagez le code de la room avec vos amis.
4. Dessinez et voyez les dessins des autres en temps réel !
